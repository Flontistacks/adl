# Tech Stack

## Overview

`adl` is a macOS terminal download manager that wraps aria2c with an interactive TUI (Mole-style). Single Go binary, English UI, local build first with Homebrew distribution later.

## Core Technologies

### Language

**Primary**: Go
**Version**: 1.22+

### Framework

**Framework**: Bubble Tea (TUI) + Cobra (CLI) + Lip Gloss (styling)
**Rationale**: Standard stack for interactive terminal apps; Cobra provides `--help` on all commands

### Runtime

**Runtime**: Native binary (no runtime dependency beyond aria2c)
**Version**: N/A

## Data Layer

Not applicable — no database. Configuration stored in YAML file.

### Configuration

**Location**: `~/.config/adl/config.yaml`
**Contents**: Default download directory, aria2c path, optional speed limit
